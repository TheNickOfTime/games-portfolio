Pac Man: Part 3

NickOctober 20, 2017October 20, 2017

I suppose I will never be done working on this project. Since the last update, I have done quite a few things. The most noticeable difference is that the enemy AI now does some rudimentary, custom pathfinding towards the player, making gameplay more challenging and interesting. Additionally, I rewrote the player and enemy controllers to derive from the same master controller class, I have been wanting to do this for a while now. Other than being easier to work with, it makes the implementation of a rewind time mechanic a lot simpler. The rewind time mechanic is also a new feature, and I will be working on it for the next ~2 weeks. Its functional, but there is a lot more that it could be doing.
